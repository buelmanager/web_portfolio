### 4. Features (프리미엄 자산관리 서비스)
**목적**: 8가지 핵심 서비스 소개

**구성 요소**:
- 섹션 제목: "프리미엄 자산관리 서비스"
- 서브 카피: "초고액 자산가를 위한 맞춤형 웰스 매니지먼트 / 글로벌 네트워크와 전문가 팀이 함께합니다."
- 카드 개수: 8개 (4열 그리드)

**콘텐츠 목록**:
1. **맞춤형 포트폴리오** (TrendingUp) - 당신만의 투자 전략
2. **글로벌 자산배분** (Globe) - 전세계 시장 접근
3. **세무 최적화** (PieChart) - 절세 전략 컨설팅
4. **부동산 투자** (Briefcase) - 프리미엄 자산 발굴
5. **상속 설계** (Users) - 세대 간 자산 이전
6. **전담 PB 배정** (Award) - 1:1 프라이빗 서비스
7. **대체투자** (Shield) - PE, VC, 헤지펀드 접근
8. **보안 & 프라이버시** (Lock) - 철저한 기밀 보장

**특징**:
- 다크 서피스 배경 (Surface)
- 각 카드는 Surface Light 배경에 Border 테두리
- 아이콘은 골드 배경 (secondary/10) 위 Lucide React 아이콘
- 호버 시 카드 상승 효과 (hover-lift: translateY -4px)
- 호버 시 제목 골드 컬러 변경
- 호버 시 아이콘 배경 진하게 (secondary/20)
- FadeInSection으로 스크롤 시 순차 등장 (0.1초 간격)
- 반응형: 모바일 1열, 태블릿 2열, 데스크톱 4열

### 5. FAQ (자주 묻는 질문)
**목적**: 고객 궁금증 해소 및 신뢰 구축

**구성 요소**:
- 섹션 제목: "자주 묻는 질문"
- 아코디언 형식 5개 항목

**콘텐츠 목록**:
1. **최소 가입 자산은?**
   - 100억원 이상 금융자산 보유 고객 대상
   - 플래티넘, 다이아몬드, 블랙 등급별 차별화 서비스

2. **PB 배정 기준은?**
   - 투자 성향, 자산 규모, 선호 투자 분야 종합 분석
   - 15년 이상 경력, 글로벌 자격증 보유 전문가

3. **수수료 구조는?**
   - 운용 규모와 성과에 따라 차등 적용
   - 투명한 보수 체계, 숨겨진 비용 없음
   - 1:1 상담 시 상세 안내

4. **해외 자산도 관리 가능한가요?**
   - 미국, 유럽, 아시아 글로벌 네트워크
   - 해외 부동산, 증권, 대체투자 자산 통합 관리
   - 현지 전문가 네트워크 협력

5. **상담은 어떻게 진행되나요?**
   - 온라인 예약 후 전담 매니저 연락
   - 서울 강남 본점 또는 고객 희망 장소에서 1:1 대면
   - 정기 리뷰와 수시 소통

**특징**:
- 다크 서피스 배경
- 아코디언 버튼 클릭 시 펼침/접힘
- 첫 번째 항목 기본 열림 상태
- ChevronDown 아이콘 회전 애니메이션 (180도)
- 호버 시 배경 변화 (background/30)
- FadeInSection으로 순차 등장 (0.1초 간격)
- 최대 높이 애니메이션 (max-h-0 → max-h-96)

## 🎬 인터랙션 & 애니메이션

### 스크롤 애니메이션
- **TextReveal**: 텍스트가 3D 회전하며 등장
  - 초기 상태: opacity 0, y 50px, rotateX -90도
  - 최종 상태: opacity 1, y 0, rotateX 0도
  - 지속 시간: 0.8초, stagger 0.03초
  - ScrollTrigger: 85% 지점에서 시작

- **FadeInSection**: 섹션이 아래에서 위로 페이드인
  - 방향 설정 가능 (up, down, left, right)
  - 딜레이 설정 가능
  - ScrollTrigger 연동

- **SectionTransition**: 섹션 전환 시 부드러운 트랜지션
  - GSAP ScrollTrigger 활용
  - 페이드인 + Y축 이동

- **ScrollProgress**: 페이지 상단에 스크롤 진행도 바 표시
  - 골드 그라데이션 (secondary → accent)
  - 스크롤 위치에 따라 너비 변경
  - 고정 위치 (top-0, z-index: 50)

### 호버 효과
- **버튼**: 
  - 골드 그라데이션 방향 반전
  - 스케일 확대 (1.05배)
  - 골드 그로우 효과

- **카드**: 
  - 상승 효과 (translateY -4px)
  - 골드 그림자 (0 20px 40px rgba(212, 175, 55, 0.15))
  - 제목 골드 컬러 변경
  - 아이콘 배경 진하게

- **메뉴**: 
  - 텍스트 컬러 골드로 변경 (text-secondary → secondary)

- **링크**: 
  - 골드 컬러 변경 (transition-colors)

### 특수 효과
- **MagneticButton**: 마우스 위치에 따라 버튼이 자석처럼 끌림
  - 마우스 이동 시: x, y 좌표 계산 (0.3배 감쇠)
  - 마우스 떠날 시: elastic ease로 원위치
  - GSAP 애니메이션 활용

- **Glass Effect**: 반투명 배경 + 블러 효과
  - background: rgba(26, 41, 66, 0.85)
  - backdrop-filter: blur(12px)

- **Gold Glow**: 골드 컬러 그림자 효과
  - box-shadow: 0 0 40px rgba(212, 175, 55, 0.2)

- **Hover Lift**: 호버 시 카드 상승 + 그림자 강화
  - transform: translateY(-4px)
  - box-shadow: 0 20px 40px rgba(212, 175, 55, 0.15)
