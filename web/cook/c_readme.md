# master | 미식의 정점 - 웹사이트 소개

> 최상의 재료와 장인의 손길이 만나는 파인 다이닝 레스토랑

---

## 📋 프로젝트 개요

### 브랜드 정보
- **브랜드명**: master (마스터)
- **업종**: 파인 다이닝 레스토랑
- **컨셉**: 미식의 정점, 프렌치 코스 요리, 럭셔리 다이닝
- **타겟**: 특별한 날을 기념하는 고객, 미식가, 비즈니스 미팅, 기념일 고객

### 사이트 목적
- 레스토랑의 프리미엄 브랜드 이미지 구축
- 시그니처 메뉴 및 코스 요리 소개
- 온라인 예약 유도 및 고객 문의 대응
- 파인 다이닝 경험의 시각적 전달

## 🔍 섹션별 상세 설명

### 1. Preloader (프리로더)
**목적**: 페이지 로딩 중 브랜드 인상 강화

**구성 요소**:
- 브랜드 로고 또는 로딩 애니메이션
- 부드러운 페이드아웃 효과

**특징**:
- 첫 방문 시 브랜드 아이덴티티 각인
- 로딩 완료 후 자연스러운 메인 화면 전환

---

### 2. Header (헤더)
**목적**: 사이트 네비게이션 및 브랜드 아이덴티티

**구성 요소**:
- 로고: "master" (mas는 Primary 컬러, ter는 검정)
- 메인 메뉴: 홈, 스토리, 메뉴, 예약
- CTA 버튼: "예약하기"

**특징**:
- 스크롤 시 배경이 투명에서 흰색으로 변화
- 모바일에서는 햄버거 메뉴로 전환
- 고정 헤더로 항상 접근 가능
- 스크롤 위치에 따라 텍스트 색상 자동 변경

---

### 3. Hero Section (메인 비주얼)
**목적**: 첫인상 & 핵심 메시지 전달

**구성 요소**:
- 배경: 파인 다이닝 영상 (Pexels 고품질 영상) 또는 대체 이미지
- 메인 카피: "미식의 정점을 경험하세요."
- 서브 카피: "master Fine Dining Experience"
- 통계/강조 정보: "10년 전통 · 시즌 메뉴", "코스 메뉴"
- CTA 버튼: "메뉴 보기", "예약하기"

**특징**:
- 전체 화면 크기의 임팩트 있는 비주얼
- 텍스트 순차 등장 애니메이션 (TextReveal)
- 어두운 그라데이션 오버레이로 텍스트 가독성 확보
- 하단 스크롤 유도 아이콘 (바운스 애니메이션)
- 영상 로딩 실패 시 자동으로 이미지로 대체

---

### 4. Features (특별한 경험)
**목적**: 레스토랑의 핵심 서비스 및 차별점 소개

**구성 요소**:
- 섹션 제목: "특별한 경험을 선사합니다"
- 카드 개수: 8개
- 각 카드 구성:
  - 아이콘 (Lucide React)
  - 제목
  - 설명

**콘텐츠 목록**:
1. **시즌 코스 메뉴** - 제철 최상의 재료로 구성된 프리미엄 코스
2. **프라이빗 다이닝** - 특별한 날을 위한 독립된 공간
3. **와인 페어링** - 소믈리에가 추천하는 완벽한 매칭
4. **신선한 재료** - 매일 공수되는 최상급 식재료
5. **런치 & 디너** - 점심 코스부터 풀코스 디너까지
6. **미슐랭 경력 셰프** - 세계 유수의 레스토랑 경력
7. **시그니처 디저트** - 파티시에의 예술적 창작품
8. **프라이빗 이벤트** - 기업 행사, 웨딩, 가족 모임

**특징**:
- 4열 그리드 레이아웃 (모바일: 1열, 태블릿: 2열)
- 호버 시 카드가 위로 떠오르는 효과 (hover-lift)
- 순차적 페이드인 애니메이션 (각 0.1초 간격)
- 아이콘 배경색 호버 시 변화

---

### 5. Product Gallery (시그니처 메뉴)
**목적**: 대표 메뉴 비주얼 소개

**구성 요소**:
- 섹션 제목: "시그니처 메뉴"
- 서브 카피: "셰프의 철학과 최상의 재료가 만나는 순간"
- 캐러셀 슬라이더 (Swiper)

**콘텐츠 목록**:
1. **한우 안심 스테이크** - 최상급 한우 안심과 트러플 소스의 조화
2. **랍스터 비스크** - 신선한 랍스터로 우려낸 깊은 풍미
3. **시즌 코스 메뉴** - 셰프의 철학이 담긴 8가지 코스
4. **와규 타르타르** - 캐비어와 함께 즐기는 프리미엄 타르타르
5. **시그니처 디저트** - 파티시에의 예술적 창작 디저트

**특징**:
- Swiper 캐러셀로 좌우 스와이프 가능
- 반응형: 모바일 1개, 태블릿 2개, 데스크톱 3개 동시 표시
- 이미지 호버 시 확대 효과 (scale-110)
- 네비게이션 화살표 및 페이지네이션 제공
- 어두운 그라데이션 오버레이 호버 효과

### 6. Category Gallery (다이닝 경험)
**목적**: 다양한 다이닝 카테고리 시각적 소개

**구성 요소**:

- 서브 카피: "당신만을 위한 특별한 미식 여정을 준비합니다"

**콘텐츠 목록**:

3. **와인 셀렉션** - 150+ 종

5. **애피타이저** - 셰프 추천

**특징**:
- 정사각형 이미지 레이아웃
- 이미지 위에 텍스트 오버레이
- 반응형: 모바일 2개, 태블릿 3개, 데스크톱 4~5개

**목적**: 고객 문의 사항 사전 해결

**구성 요소**:
- 섹션 제목: "자주 묻는 질문"

**콘텐츠 목록**:
1. **예약은 어떻게 하나요?**
   - 온라인 예약 또는 전화 예약 안내
   - 예약 가능 시간 및 사전 예약 권장 사항

   - 스마트 캐주얼 권장

   - 무료 발렛파킹 서비스 제공
   - 이용 방법 안내

4. **코르키지가 가능한가요?**
   - 병당 30,000원
   - 제한 사항 안내

5. **특별한 날 서프라이즈가 가능한가요?**
   - 케이크 및 데코레이션 서비스

**특징**:

## 🎬 인터랙션 & 애니메이션

### 스크롤 애니메이션
- **Fade In**: 모든 섹션이 스크롤 시 부드럽게 페이드인 (FadeInSection 컴포넌트)
- **Text Reveal**: 제목 텍스트가 순차적으로 등장 (TextReveal 컴포넌트)
- **Stagger**: Features 카드들이 0.1초 간격으로 순차 등장
- **Direction**: 좌우에서 슬라이드 인 효과 (Category Selector)

### 호버 효과
- **버튼**: 배경색 변화 및 스케일 확대 (scale-105)
- **카드**: 위로 떠오르는 효과 (hover-lift, translateY -4px)
- **이미지**: 확대 효과 (scale-110) 및 오버레이 표시
- **링크**: Primary 컬러로 변경

### 특수 효과
- **Glass Effect**: 헤더의 반투명 배경 및 블러 효과
- **Gradient Overlay**: Hero 섹션의 어두운 그라데이션
- **Bounce Animation**: Hero 하단 스크롤 유도 아이콘
- **Accordion**: FAQ 아코디언 열림/닫힘 애니메이션
- **Tab Transition**: Category Selector 탭 전환 애니메이션

## 💻 기술 스택 (간단 요약)

### 프론트엔드
- **프레임워크**: Next.js 16.1.6 (React 19.2.3 기반)
- **스타일링**: Tailwind CSS 4
- **애니메이션**: GSAP 3.14.2 (@gsap/react 2.1.2)
- **캐러셀**: Swiper 12.1.0
- **아이콘**: Lucide React 0.563.0

### 주요 기능
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 완벽 대응
