## 📋 프로젝트 개요

**master**는 프리미엄 파인 다이닝 레스토랑을 위한 웹사이트입니다. 고급스러운 브랜드 아이덴티티와 세련된 사용자 경험을 통해 미식의 정점을 표현합니다.

### 🎯 브랜드 정보

- **브랜드명**: master
- **컨셉**: 미식의 정점
- **타겟**: 파인 다이닝, 프렌치 레스토랑, 미슐랭 경력 셰프
- **키워드**: 마스터, 파인다이닝, 레스토랑, 미슐랭, 코스요리, 프렌치

## 🎬 애니메이션 & 인터랙션

### GSAP 애니메이션

#### 1. **TextReveal** 컴포넌트
- 텍스트를 글자 단위로 분해
- 3D 회전 효과 (rotateX: -90 → 0)
- 페이드인 + 위로 슬라이드 (y: 50 → 0)
- Stagger 효과 (0.03초 간격)
- ScrollTrigger 연동 (화면 85% 지점에서 트리거)

#### 2. **FadeInSection** 컴포넌트
- 4방향 페이드인 지원 (up, down, left, right)
- 기본값: 위로 슬라이드 (y: 60 → 0)
- 1초 duration, power3.out easing
- ScrollTrigger 연동

### CSS 애니메이션

#### 1. **Hover Effects**
- `.hover-lift`: 카드 hover 시 위로 이동 + 그림자 효과
- 이미지 scale 효과 (1 → 1.1)
- 버튼 scale 효과 (1 → 1.05)

#### 2. **Bounce Animation**
- 스크롤 다운 인디케이터
- 무한 반복 bounce 애니메이션

#### 3. **Gradient Effects**
- `.text-gradient`: 텍스트 그라디언트 (primary → secondary)
- `.glass`: 유리 효과 (backdrop-filter blur)

### Swiper 인터랙션
- 네비게이션 버튼 (이전/다음)
- 페이지네이션 (점 표시)
- 터치/드래그 스와이프
- 반응형 breakpoints

## 🚀 실행 방법

### 개발 서버 실행
```bash
npm run dev
```
- 로컬 개발 서버: http://localhost:3000
- 핫 리로드 지원

### 프로덕션 빌드
```bash
npm run build
```
- 최적화된 프로덕션 빌드 생성

### 프로덕션 서버 실행
```bash
npm run start
```
- 빌드된 앱을 프로덕션 모드로 실행

### 린트 검사
```bash
npm run lint
```
- ESLint를 사용한 코드 품질 검사
