# 🏛️ vertex | 공간을 디자인하다

> 혁신적인 건축 설계로 공간의 가치를 창조합니다. 주거, 상업, 문화 공간의 새로운 기준.

---

## 📋 프로젝트 개요

**vertex**는 혁신적인 건축 설계로 공간의 가치를 창조하는 건축사무소 웹사이트입니다.  
주거, 상업, 문화 공간의 새로운 기준을 제시하며, 15년 경력과 200개 이상의 프로젝트 경험을 바탕으로 고객에게 최상의 설계 서비스를 제공합니다.

## 🎬 애니메이션 & 인터랙션

### GSAP 애니메이션
1. **TextReveal** (텍스트 리빌)
   - 글자별 순차 페이드인
   - Y축 이동 + 블러 효과
   - ScrollTrigger 연동

2. **FadeInSection** (섹션 페이드인)
   - 방향: up, down, left, right, scale
   - 스크롤 트리거 기반
   - 블러 옵션 지원

3. **CountUp** (숫자 카운트업)
   - 스크롤 진입 시 0부터 목표값까지 애니메이션
   - prefix/suffix 지원

4. **Parallax Scroll** (패럴랙스 스크롤)
   - 이미지 배경 스크롤 효과
   - CTA 섹션 배경

5. **Icon Rotation** (아이콘 회전)
   - Features 카드 호버 시 360도 회전

6. **Accordion Animation** (아코디언 애니메이션)
   - FAQ 섹션 높이 애니메이션

### 호버 효과
- **hover-lift**: 카드 호버 시 Y축 이동 + 그림자
- **border-color**: 테두리 색상 전환 (border → secondary)
- **text-color**: 텍스트 색상 전환 (primary → secondary)
- **scale**: 버튼 스케일 애니메이션

### UI 컴포넌트
- **Preloader**: 페이지 로딩 애니메이션
- **SidebarNav**: 사이드바 네비게이션
- **FloatingButton**: 플로팅 버튼
- **Carousel**: Swiper 기반 캐러셀

## 🛠️ 기술 스택

### 프레임워크 & 라이브러리
- **Next.js 16.1.6** - React 프레임워크
