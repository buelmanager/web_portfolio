# vertex | 공간을 디자인하다

> 혁신적인 건축 설계로 공간의 가치를 창조하는 건축설계사무소 웹사이트

---

## 📋 프로젝트 개요

### 브랜드 정보
- **브랜드명**: vertex (버텍스)
- **업종**: 건축설계사무소 / 공간 디자인
- **컨셉**: 혁신적, 전문성, 가치 창조
- **타겟**: 주거/상업/문화 공간 설계를 원하는 개인 및 기업 고객

### 사이트 목적
- 건축설계사무소의 전문성과 포트폴리오 소개
- 다양한 프로젝트 카테고리별 실적 전시
- 고객 상담 및 문의 유도
- 브랜드 아이덴티티 강화 및 신뢰도 구축

---

## 🎨 디자인 컨셉

### 컬러 팔레트
- **Primary**: `#2C2C2C` (Charcoal) - 전문성과 신뢰를 상징하는 차콜 그레이
- **Secondary**: `#C9A86C` (Gold) - 프리미엄과 품격을 나타내는 골드 액센트
- **Accent**: `#B8956A` (Muted Gold) - 부드러운 골드 톤
- **Background**: `#FAFAFA` (Light Gray) - 깔끔하고 모던한 배경
- **Surface**: `#FFFFFF` (Pure White) - 순백의 표면

### 디자인 스타일
- **미니멀리즘**: 깔끔하고 정제된 레이아웃으로 콘텐츠에 집중
- **프리미엄 감성**: 골드 액센트와 세련된 타이포그래피로 고급스러움 표현
- **모던 건축**: 건축적 요소를 반영한 구조적이고 균형잡힌 디자인
- **인터랙티브**: GSAP 애니메이션으로 생동감 있는 사용자 경험 제공

## 🔍 섹션별 상세 설명

### 1. Header (헤더)
**목적**: 사이트 네비게이션 및 브랜드 아이덴티티

**구성 요소**:
- 로고: "ver**tex**" (tex 부분 골드 강조)
- 메인 메뉴: 홈, 프로젝트, 서비스, 문의
- CTA 버튼: "상담 신청"
- 모바일 햄버거 메뉴

**특징**:
- 스크롤 시 배경이 투명에서 흰색 글래스모피즘으로 변화
- 스크롤 전: 투명 배경 + 흰색 텍스트
- 스크롤 후: 흰색 배경 + 검은색 텍스트 + 하단 보더
- 부드러운 트랜지션 효과 (500ms)

- 상단 라벨: "vertex Architecture & Design"

- 서브 카피: "가치를 창조하다"

**특징**:

  2. **상업 공간 설계** - 브랜드 가치를 높이는 공간
  3. **리모델링 & 인테리어** - 기존 공간의 새로운 발견
  4. **3D 시각화** - 완공 전 공간을 미리 경험

  6. **친환경 설계** - 지속 가능한 건축 솔루션

### 8. FAQ (자주 묻는 질문)
**목적**: 고객 궁금증 해소 및 신뢰도 향상

**구성 요소**:
- 섹션 제목: "자주 묻는 질문"
- 5개의 아코디언 항목:
  1. **설계 기간은 얼마나 걸리나요?**
     - 기본 설계 4-8주, 실시 설계 6-12주
  2. **설계 비용은 어떻게 산정되나요?**
     - 용도, 연면적, 설계 범위에 따라 산정, 대한건축사협회 기준 참고
  3. **시공사 연결도 가능한가요?**
     - 신뢰할 수 있는 시공사 추천 및 전 과정 조율
  4. **인허가 대행도 해주시나요?**
     - 건축허가, 용도변경, 착공신고, 사용승인 등 모든 절차 대행
  5. **3D 시각화는 언제 받을 수 있나요?**
     - 기본 설계 완료 후 2-3주 내 제공

**특징**:
- GSAP 애니메이션으로 부드러운 아코디언 효과
- 첫 번째 항목 기본 오픈
- ChevronDown 아이콘 회전 애니메이션
- 호버 시 보더 색상 변화 (border → secondary/30)
- 질문 텍스트 호버 시 색상 변화 (primary → secondary)

## 🎬 인터랙션 & 애니메이션

### 스크롤 애니메이션
- **Fade In**: 모든 섹션이 스크롤 시 페이드인 (FadeInSection 컴포넌트)
- **Parallax**: 
  - Hero 비디오 배경
  - Product Gallery 이미지 (scale 1.05)
  - Category Gallery 이미지 (scale 1.2 → 1)
  - CTA 배경 이미지 (yPercent 30)
- **Stagger**: Features 카드 순차 등장 (0.1s 간격)
- **Text Reveal**: 제목 텍스트 글자별 등장 애니메이션
- **Count Up**: 통계 숫자 카운트업 효과

### 호버 효과
- **버튼**: 
  - 배경/텍스트 색상 반전
  - 부드러운 트랜지션 (300ms)
- **카드**: 
  - 리프트 효과 (translateY -4px)
  - 그림자 증가
  - 보더 색상 변화
- **이미지**: 
  - 확대 효과 (scale 1.05 ~ 1.1)
  - 오버레이 등장
- **아이콘**: 
  - Features 섹션 아이콘 360도 회전

### 특수 효과
- **Glass Morphism**: Header 스크롤 시 글래스 효과
- **Gradient Overlay**: 이미지 위 그라디언트 오버레이
- **Shimmer**: 골드 색상 쉬머 애니메이션 (CSS 키프레임)
- **Accordion**: FAQ 아코디언 부드러운 높이 애니메이션
- **Carousel**: Swiper 슬라이드 전환 효과
