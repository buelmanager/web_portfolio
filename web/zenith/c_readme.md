## 🎬 애니메이션 & 인터랙션

### GSAP 애니메이션
1. **FadeInSection**: 스크롤 트리거 기반 페이드인 애니메이션
   - 방향: up, down, left, right
   - 딜레이 조절 가능
   - ScrollTrigger 플러그인 사용

2. **TextReveal**: 텍스트 리빌 애니메이션
   - 히어로 섹션 메인 카피에 적용
   - 딜레이 조절 가능

### CSS 애니메이션
1. **Gold Shimmer**: 골드 색상이 흐르는 텍스트 효과
   - 히어로 섹션 메인 타이틀에 적용
   - 3초 주기 무한 반복

2. **Fade In Up**: 기본 페이드인 + 상승 애니메이션
   - 0.8초 duration
   - 히어로 서브 카피에 적용

3. **Bounce**: 바운스 애니메이션
   - 히어로 하단 스크롤 인디케이터에 적용

4. **Hover Lift**: 호버 시 카드 상승 효과
   - Transform translateY(-4px)
   - 로즈 골드 그림자 효과

5. **Card 3D**: 3D 틸트 효과
   - Features 섹션 카드에 적용
   - 마우스 움직임에 따라 rotateX, rotateY 변화

6. **Gallery Zoom**: 갤러리 이미지 확대 효과
   - 호버 시 scale(1.1)
   - 그라데이션 오버레이 페이드인

### Swiper 캐러셀
- **네비게이션**: 좌우 화살표 버튼 (반투명 검정 배경)
- **페이지네이션**: 하단 도트 (활성 도트는 로즈 골드)
- **반응형 설정**:
  - 모바일: 1개 슬라이드
  - 태블릿 (640px+): 2개 슬라이드
  - 데스크톱 (1024px+): 3개 슬라이드

## 🛠️ 기술 스택

### 프레임워크 & 라이브러리

- **React 19.2.3**: UI 라이브러리
- **TypeScript 5**: 타입 안정성

- **Tailwind CSS 4**: 유틸리티 퍼스트 CSS 프레임워크

### 애니메이션
- **GSAP 3.14.2**: 고성능 애니메이션 라이브러리
- **@gsap/react 2.1.2**: GSAP React 통합
- **ScrollTrigger**: 스크롤 기반 애니메이션

### UI 컴포넌트
- **Swiper 12.1.0**: 터치 슬라이더/캐러셀
- **Lucide React 0.563.0**: 아이콘 라이브러리

### 폰트

## 🚀 실행 방법

### 개발 서버 실행
```bash
npm run dev
# 또는
yarn dev
# 또는
bun dev
```

개발 서버가 [http://localhost:3000](http://localhost:3000)에서 실행됩니다.

### 프로덕션 빌드
```bash
npm run build
npm run start
```

### 린트 검사
```bash
npm run lint
```
