# zenith - 시간을 담은 예술

> 스위스 장인정신으로 완성된 럭셔리 주얼리와 타임피스. 영원을 담은 컬렉션.

---

## 📋 프로젝트 개요

### 브랜드 정보
- **브랜드명**: zenith (제니스)
- **업종**: 럭셔리 주얼리 & 스위스 타임피스
- **컨셉**: 타임리스, 장인정신, 영원한 가치
- **타겟**: 프리미엄 주얼리 및 시계 애호가, 럭셔리 라이프스타일 추구 고객

### 사이트 목적
- 스위스 장인정신과 150년 헤리티지를 강조한 브랜드 아이덴티티 구축
- 다이아몬드, 타임피스, 브라이덜 컬렉션 등 프리미엄 제품 라인업 소개
- 프라이빗 부티크 예약 및 맞춤 제작 상담 유도
- 럭셔리 브랜드에 걸맞은 고급스러운 온라인 경험 제공

## 🎬 인터랙션 & 애니메이션

### 스크롤 애니메이션
- **FadeInSection**: 모든 주요 섹션에 적용
  - 방향: up (기본), down, left, right
  - GSAP ScrollTrigger 사용
  - 85% 뷰포트 진입 시 트리거
  - 1초 duration, power3.out easing
  - 순차 등장 (delay 옵션)

- **TextReveal**: 제목 텍스트 애니메이션
  - 글자별 순차 등장
  - 골드 시머 효과와 결합

- **Parallax**: CTA 섹션 배경
