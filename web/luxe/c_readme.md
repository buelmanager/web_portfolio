# 🏢 DCC&P | 디지털 라이프스타일의 정점

> 2,000여 종의 프리미엄 IT 하드웨어 & 가전. 25년 노하우의 디지털 커머스 플랫폼.

---

## 📋 프로젝트 개요

**DCC&P**(디씨씨앤피)는 IT 하드웨어부터 생활가전까지 2,000여 종의 프리미엄 제품을 제공하는 디지털 커머스 플랫폼입니다. 25년간 축적된 노하우를 바탕으로 Dell, HP 등 글로벌 브랜드의 워크스테이션, 서버, 가전제품을 B2B/B2C 고객에게 제공합니다.

### 🎯 핵심 가치
- **프리미엄 제품 라인업**: Dell PRECISION, HP ProLiant 등 전문가급 IT 하드웨어
- **25년 노하우**: 검증된 재고 관리 및 품질 보증 시스템
- **B2B 전문 서비스**: 대량 구매, 세금계산서, 맞춤 견적
- **중고/리퍼비시**: 최대 70% 할인의 가성비 제품

## ✨ 애니메이션 & 인터랙션

### GSAP 애니메이션

#### 1. **TextReveal** (텍스트 등장 효과)
- **효과**: 글자별 3D 회전 + 페이드인
- **속성**: 
  - `opacity: 0 → 1`
  - `y: 50 → 0`
  - `rotateX: -90 → 0`
- **타이밍**: 0.8초 duration, 0.03초 stagger
- **트리거**: ScrollTrigger (화면 85% 진입 시)
- **사용 위치**: Hero, Features, Product Gallery, Category Gallery, FAQ, CTA 헤드라인

#### 2. **FadeInSection** (섹션 등장 효과)
- **방향**: up / down / left / right
- **효과**: 방향별 슬라이드 + 페이드인
- **타이밍**: 1초 duration
- **트리거**: ScrollTrigger (화면 85% 진입 시)
- **사용 위치**: 모든 섹션 콘텐츠

### CSS 애니메이션

#### 1. **Hover Lift**
- **효과**: 카드 상승 + 그림자 강화
- **속성**: `transform: translateY(-4px)`, `box-shadow: 0 20px 40px rgba(0, 102, 204, 0.2)`
- **사용 위치**: Features 카드

#### 2. **Image Scale**
- **효과**: 이미지 1.1배 확대
- **타이밍**: 0.5초 transition
- **사용 위치**: Product Gallery, Category Gallery

#### 3. **Bounce Animation**
- **효과**: 상하 바운스
- **사용 위치**: Hero 스크롤 인디케이터

#### 4. **Glass Morphism**
- **효과**: 반투명 배경 + 블러
- **속성**: `background: rgba(17, 17, 17, 0.8)`, `backdrop-filter: blur(10px)`
- **사용 위치**: Header (스크롤 시)

### Swiper 인터랙션
- **Navigation**: 좌우 화살표 버튼 (원형, 반투명 배경)
- **Pagination**: 동적 불릿 (활성 시 Primary 컬러)
- **Autoplay**: 선택적 자동 재생
- **Loop**: 무한 루프
- **Breakpoints**: 반응형 슬라이드 개수 조정

## 🚀 실행 방법

### 1. 의존성 설치
```bash
npm install
```

### 2. 개발 서버 실행
```bash
npm run dev
```
- 브라우저에서 `http://localhost:3000` 접속

### 3. 프로덕션 빌드
```bash
npm run build
```

### 4. 프로덕션 서버 실행
```bash
npm start
```

### 5. 린트 검사
```bash
npm run lint
```
