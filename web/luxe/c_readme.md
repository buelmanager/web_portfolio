# DCC&P - 디지털 라이프스타일의 정점

> 25년 노하우로 제공하는 2,000여 종의 프리미엄 IT 하드웨어 & 가전 커머스 플랫폼

## 📋 프로젝트 개요

### 브랜드 정보
- **브랜드명**: DCC&P (디씨씨앤피)
- **업종**: IT 하드웨어 & 가전 B2B/B2C 커머스
- **컨셉**: 프리미엄, 전문성, 신뢰
- **타겟**: 기업 고객(B2B), 일반 소비자(B2C), IT 전문가

### 사이트 목적
- 2,000여 종의 프리미엄 IT 하드웨어 및 가전 제품 소개
- Dell, HP 등 글로벌 브랜드 제품 라인업 전시
- B2B 대량 구매 및 맞춤 견적 서비스 제공
- 25년 노하우를 바탕으로 한 전문 상담 및 솔루션 제안

## 🔍 섹션별 상세 설명

## 🎬 인터랙션 & 애니메이션

### 스크롤 애니메이션
- **Fade In**: 모든 섹션이 스크롤 시 페이드인 (FadeInSection 컴포넌트)
- **Text Reveal**: 메인 타이틀이 글자별로 순차 등장 (TextReveal 컴포넌트)
- **Stagger**: Features 카드가 순차적으로 등장 (0.1초 간격)
- **Direction**: 좌우 슬라이드 애니메이션 (Category Selector)

### 호버 효과
- **버튼**: 배경색 변화, 확대 효과 (scale-105)
- **카드**: 위로 떠오르는 효과 (hover-lift), 그림자 강화
- **이미지**: 확대 효과 (scale-110), 오버레이 투명도 변화
- **텍스트**: 색상 변화 (gray → primary blue)

### 특수 효과
- **Glass Morphism**: 헤더 스크롤 시 반투명 배경 + backdrop-blur
- **Gradient Text**: 일부 텍스트에 블루 그라데이션 적용
- **Bounce**: Hero 섹션 하단 스크롤 다운 아이콘
- **Carousel**: Swiper 라이브러리로 부드러운 슬라이드 전환

## 💻 기술 스택 (간단 요약)

### 프론트엔드
- **프레임워크**: Next.js 16.1.6 (React 19.2.3 기반)
- **스타일링**: Tailwind CSS 4
- **애니메이션**: GSAP (@gsap/react 2.1.2), Swiper 12.1.0
- **아이콘**: Lucide React 0.563.0
- **폰트**: Geist Sans, Geist Mono (Google Fonts)

### 주요 기능
- **반응형 디자인**: 모바일/태블릿/데스크톱 완벽 대응
- **SEO 최적화**: 메타데이터, Open Graph 설정 완료
