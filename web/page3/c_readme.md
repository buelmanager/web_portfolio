# MAISON ÉLISE - 럭셔리 레더 굿즈 웹사이트

> 파리의 우아함을 담은 프리미엄 가죽 핸드백 브랜드 웹사이트

---

## 📋 프로젝트 개요

### 브랜드 정보
- **브랜드명**: MAISON ÉLISE (메종 엘리제)
- **업종**: 럭셔리 레더 굿즈 / 프리미엄 핸드백
- **컨셉**: 파리지앵 우아함, 장인정신, 타임리스 엘레강스
- **타겟**: 럭셔리 패션을 추구하는 세련된 여성 고객층

### 사이트 목적
- 100년 전통의 파리 장인정신을 시각적으로 전달
- 프리미엄 핸드백 컬렉션을 감각적으로 소개
- 브랜드 헤리티지와 제작 과정을 스토리텔링
- 고급스러운 사용자 경험을 통한 브랜드 가치 강화

## 🔍 섹션별 상세 설명

### 1. Header (헤더)
**목적**: 브랜드 아이덴티티 강조 및 직관적인 네비게이션

**구성 요소**:
- 중앙 로고: "MAISON ÉLISE" (2단 구성)
- 좌측 메뉴: Collections, Savoir-Faire, La Maison
- 우측 메뉴: Boutiques, Journal, Contact (CTA)
- 모바일 햄버거 메뉴

**특징**:
- 스크롤 시 배경색 변화 (투명 → 블랙)
- 부드러운 페이드 인/아웃 효과
- 모바일 전체화면 메뉴

---

### 2. Video Hero (메인 비주얼)
**목적**: 강렬한 첫인상과 브랜드 메시지 전달

**구성 요소**:
- 배경: 핸드백 제작 과정 영상 (자동 재생, 루프)
- 메인 카피: "The Art of Timeless Elegance"
- 서브 카피: "Parisian Leather Craftsmanship Since 1923"
- 스크롤 인디케이터: "Scroll to Explore"

**특징**:
- 영상 줌 인 애니메이션 (1.3배 → 1배)
- 타이틀 3D 회전 등장 효과 (rotateX)
- 스크롤 시 패럴랙스 효과 (영상 확대, 텍스트 페이드 아웃)
- 다크 오버레이로 텍스트 가독성 확보

---

### 3. Zoom Section (줌 인 스토리텔링)
**목적**: 장인정신과 디테일에 대한 집중

**구성 요소**:
- 배경: 가죽 제작 과정 이미지
- 메인 텍스트: "Every stitch tells a story"
- 서브 텍스트: "of unparalleled artistry"

**특징**:
- 스크롤 기반 이미지 줌 아웃 (2.5배 → 1배)
- Clip-path 애니메이션으로 이미지 확장
- 단어별 순차 등장 효과 (stagger)
- 섹션 고정(pin) 효과로 몰입감 강화

---

### 4. Collection Grid (컬렉션 그리드)
**목적**: 4가지 핸드백 컬렉션 소개

**구성 요소**:
- 섹션 제목: "Collections" (글자별 애니메이션)
- 소개 문구: "Discover our exceptional leather creations..."
- 4개 컬렉션 카드:
  1. **Étoile** - Evening Collection (2024)
  2. **Parisienne** - Day-to-Night Totes (2024)
  3. **Héritage** - Classic Satchels (2023)
  4. **Voyage** - Travel Essentials (2024)
- CTA 버튼: "View All Collections"

**특징**:
- 비대칭 그리드 레이아웃 (다양한 이미지 비율)
- 카드별 패럴랙스 이미지 효과 (위/아래 방향 교차)
- 호버 시 이미지 확대 및 오버레이 효과
- 스크롤 기반 순차 등장 애니메이션

---

### 5. Horizontal Scroll (브랜드 히스토리)
**목적**: 100년 브랜드 역사를 타임라인으로 시각화

**구성 요소**:
- 섹션 제목: "Notre Histoire" (우리의 역사)
- 5개 타임라인 카드 (가로 스크롤):
  - **1923** - Les Débuts (창립)
  - **1952** - L'Étoile (시그니처 클래스프 탄생)
  - **1978** - Global Recognition (글로벌 확장)
  - **1998** - Heritage Revival (헤리티지 컬렉션)
  - **2024** - Sustainable Future (지속가능한 럭셔리)

**특징**:
- 세로 스크롤 → 가로 스크롤 전환
- 섹션 고정(pin) 후 컨텐츠 수평 이동
- 각 카드 이미지 줌 아웃 효과
- 하단 프로그레스 바로 진행 상황 표시

---

### 6. Parallax Showcase (패럴랙스 쇼케이스)
**목적**: 제품 디테일을 예술적으로 표현

**구성 요소**:
- 5개 레이어 이미지 (다양한 핸드백 디테일)
- 중앙 텍스트: "Elegance is not a trend, it's a signature."

**특징**:
- 각 레이어마다 다른 속도의 패럴랙스 효과
- 이미지 배치: 비대칭 레이아웃으로 역동성 표현
- 텍스트 단어별 페이드 인 효과
- 깊이감 있는 3D 레이어 연출

---

### 7. Manufacture (제작 과정)
**목적**: 장인의 손길과 제작 과정 강조

**구성 요소**:
- 배경: 가죽 재봉 과정 영상
- 라벨: "L'Atelier" (아틀리에)
- 제목: "Where Elegance is Crafted by Hand"
- 통계 정보:
  - **100+ Years** of Savoir-Faire
  - **38 Master** Artisans
  - **120 Hours** per Creation
  - **1 Pursuit** of Perfection

**특징**:
- 스크롤 시 영상 확대 및 모서리 둥글기 제거
- 통계 숫자 순차 등장 애니메이션
- 다크 오버레이로 텍스트 강조
- 골드 컬러 CTA 버튼

## 🎬 인터랙션 & 애니메이션

### 스크롤 애니메이션
- **Smooth Scroll**: Lenis 라이브러리로 실크처럼 부드러운 스크롤
- **Parallax**: 다층 이미지 패럴랙스 효과
- **Zoom Effects**: 스크롤 기반 이미지 줌 인/아웃
- **Horizontal Scroll**: 세로 스크롤을 가로 이동으로 전환
- **Pin Sections**: 특정 섹션 고정 후 애니메이션 재생

### 텍스트 애니메이션
- **Character Split**: 글자별 순차 등장 (stagger)
- **Word Reveal**: 단어별 페이드 인 효과
- **3D Rotation**: rotateX를 활용한 입체 등장
- **Opacity Transition**: 스크롤 기반 투명도 변화

### 이미지 효과
- **Scale Animation**: 확대/축소 애니메이션
- **Clip-path**: 마스크 효과로 이미지 공개
- **Parallax Layers**: 다층 패럴랙스 효과
- **Hover Effects**: 호버 시 확대 및 밝기 변화

### 비디오 효과
- **Auto-play Loop**: 자동 재생 및 무한 반복
- **Scroll-based Scale**: 스크롤에 따른 확대/축소
- **Border Radius Transition**: 모서리 둥글기 변화

## 💻 기술 스택 (간단 요약)

### 프론트엔드
- **프레임워크**: React 19.2.0
- **빌드 도구**: Vite 7.2.4
- **스타일링**: CSS3 (Custom Properties)
- **애니메이션**: GSAP 3.14.2 + ScrollTrigger
- **스무스 스크롤**: Lenis 1.3.17

### 주요 기능
- 반응형 디자인 (모바일/태블릿/데스크톱)
- 고성능 스크롤 애니메이션
- 비디오 자동 재생 및 최적화
- 부드러운 페이지 전환
- 접근성 고려 (시맨틱 HTML, ARIA 레이블)

### 타이포그래피
- **Display Font**: Playfair Display (우아한 세리프)
- **Body Font**: Inter (모던 산세리프)

### 개발 환경
- **패키지 매니저**: npm
- **린터**: ESLint
- **개발 서버**: Vite Dev Server (HMR 지원)

## 📊 페이지 성능

- **로딩 속도**: Vite 번들링으로 최적화된 코드 스플리팅
- **이미지 최적화**: Unsplash CDN 활용 (자동 포맷 변환 및 리사이징)
- **비디오 최적화**: Mixkit 비디오 CDN 활용
- **애니메이션 성능**: GSAP의 GPU 가속 활용
- **스크롤 성능**: Lenis의 최적화된 스무스 스크롤
- **반응형**: 3단계 브레이크포인트 (모바일 < 768px, 태블릿 768-1023px, 데스크톱 1024px+)
