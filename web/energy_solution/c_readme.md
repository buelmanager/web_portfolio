### 8. FAQ (자주 묻는 질문)
**목적**: 고객 의문 해소 및 전환율 향상

**구성 요소**:
- 섹션 제목: "자주 묻는 질문"
- 아코디언 형식 5개 질문

**콘텐츠 목록**:
1. **태양광 설치 비용은 얼마인가요?**
   - 가정용 3kW 기준 500~700만원, 정부 보조금 최대 175만원 지원
   
2. **정부 보조금은 어떻게 받나요?**
   - 보조금 신청부터 승인까지 전 과정 대행, 지자체별 추가 지원 안내
   
3. **설치 후 유지보수는 어떻게 하나요?**
   - 24시간 원격 모니터링, 연 2회 무료 정기 점검, 24시간 긴급 출동
   
4. **투자 대비 수익은 어느 정도인가요?**
   - 5~7년 내 투자비 회수, 이후 15~20년 순수익, 상업용 연 15% 이상 수익률
   
5. **태양광 패널 수명은 얼마나 되나요?**
   - 25~30년 수명, 25년 출력 보증(80% 이상), Tier-1 등급 패널 사용

**특징**:
- 클릭 시 아코디언 펼침/접힘
- 첫 번째 질문 기본 오픈

## 🎬 인터랙션 & 애니메이션

### 스크롤 애니메이션
- **FadeInSection**: 모든 섹션이 스크롤 시 페이드인 (Intersection Observer 사용)
- **TextReveal**: 메인 타이틀이 글자 단위로 순차 등장
- **Stagger**: Features 카드가 0.1초 간격으로 순차 등장
- **Parallax**: Hero 섹션 배경 영상 고정 효과

### 호버 효과
- **버튼**: 배경색 변경, 스케일 확대 (1.05배)
- **카드**: 상승 효과 (translateY -4px), 그린 그림자
- **이미지**: 확대 효과 (scale 1.1), 오버레이 투명도 변화
- **텍스트**: 그린 컬러로 변경

### 특수 효과
- **Glass Morphism**: 헤더 스크롤 시 블러 배경
