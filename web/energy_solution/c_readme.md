# ⚡ 에너지 솔루션 - 태양광 발전 및 신재생에너지 전문기업

> 지속 가능한 미래를 위한 에너지 혁신 - Green Energy Partner

---

## 📋 프로젝트 개요

**에너지 솔루션**은 태양광 발전, ESS, 신재생에너지 토탈 솔루션을 제공하는 전문기업의 웹사이트입니다.  
15년 노하우의 설치부터 유지보수까지 원스톱 서비스를 제공하며, 1,000+ 설치 실적을 보유하고 있습니다.

### 브랜드 정보
- **회사명**: (주)에너지 솔루션
- **슬로건**: 지속 가능한 미래를 위한 에너지 혁신
- **핵심 가치**: 탄소중립 실현, 친환경 에너지, 15년 노하우
- **주요 서비스**: 태양광 발전, ESS, 신재생에너지, 전력 계통 연계, 유지보수

## 📐 페이지 구조 및 섹션

### 1. 🎬 **Preloader (프리로더)**
- 페이지 로딩 시 표시되는 로딩 애니메이션
- 브랜드 로고 또는 로딩 인디케이터

### 2. 🧭 **Header (헤더)**
- 네비게이션 메뉴
- 로고 및 브랜드명
- 스크롤 시 고정 헤더

### 3. 🌟 **Hero (히어로 섹션)**
- **배경**: 태양광 패널 비디오 (fallback: 이미지)
- **메인 카피**: "지속 가능한 미래를 위한 에너지 혁신"
- **서브 카피**: "Energy Solution - Green Energy Partner"
- **통계**: 15년 노하우 · 1,000+ 설치 실적
- **CTA 버튼**: "솔루션 둘러보기", "무료 상담 신청"
- **애니메이션**: TextReveal (텍스트 순차 등장), 바운스 스크롤 아이콘

### 4. ✨ **Features (특징 섹션)**
- **제목**: "토탈 에너지 솔루션"
- **8가지 핵심 서비스**:
  1. 태양광 발전 시스템 (Sun)
  2. ESS 에너지 저장 (Battery)
  3. 전력 계통 연계 (Zap)
  4. 산업용 발전소 (Building2)
  5. 친환경 인증 (Leaf)
  6. 유지보수 서비스 (Settings)
  7. 장기 보증 (Shield)
  8. 전문 컨설팅 (Headphones)
- **레이아웃**: 4열 그리드 (반응형)
- **애니메이션**: FadeInSection (순차 페이드인), hover-lift (호버 시 상승)

### 5. 🖼️ **ProductGallery (제품 갤러리)**
- **제목**: "신재생에너지 솔루션"
- **5가지 제품**:
  1. 주택용 태양광 시스템 (3kW~10kW)
  2. 상업용 태양광 발전소
  3. 산업용 태양광 플랜트
  4. ESS 에너지 저장 시스템
  5. 풍력 발전 시스템
- **UI**: Swiper 캐러셀 (네비게이션, 페이지네이션)
- **애니메이션**: 이미지 호버 시 스케일 확대, 그라디언트 오버레이

### 6. 🏷️ **CategoryGallery (카테고리 갤러리)**
- **제목**: "솔루션 카테고리"
- **6가지 카테고리**:
  1. 주택용 태양광 (500+ 설치)
  2. 상업용 태양광 (300+ 설치)
  3. 산업용 발전소 (150+ 설치)
  4. ESS 시스템 (100+ 설치)
  5. 풍력 발전 (50+ 설치)
  6. 유지보수 서비스 (1,000+ 계약)
- **UI**: Swiper 캐러셀 (5열 그리드)
- **애니메이션**: 이미지 호버 시 스케일 확대, 오버레이 투명도 변화

### 7. 🔍 **CategorySelector (카테고리 선택기)**
- 카테고리별 필터링 기능
- 인터랙티브 선택 UI

### 8. ❓ **FAQ (자주 묻는 질문)**

### 인터랙션 효과

- **버튼 호버**: 스케일 확대, 배경색 변경
- **스크롤 아이콘**: 바운스 애니메이션 (Hero)

- **반응형 브레이크포인트**: 모바일(1열) → 태블릿(2~3열) → 데스크톱(3~5열)
