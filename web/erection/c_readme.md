**구성 요소**:

## 🎬 인터랙션 & 애니메이션

### 스크롤 애니메이션
- **Fade In**: 모든 섹션이 스크롤 시 부드럽게 나타남 (GSAP ScrollTrigger)
- **Direction Fade**: 상하좌우 방향에서 슬라이드 인 (up, down, left, right)
- **Text Reveal**: 글자별 순차 등장 애니메이션 (3D 회전 효과)
- **Stagger**: 카드 요소들이 순차적으로 등장 (0.1초 간격)

### 호버 효과
- **버튼**: 배경색 변경, 스케일 업 (1.05배)
- **카드**: 위로 리프트 효과 (-4px), 그림자 강화, 텍스트 컬러 변경
- **이미지**: 확대 효과 (1.1배), 오버레이 밝기 변경
- **링크**: Primary 컬러로 변경

### 특수 효과
- **Glassmorphism**: 헤더 스크롤 시 반투명 배경 + 블러 효과
- **Video Background**: Hero 섹션 YouTube 영상 자동 재생
- **Parallax**: 배경 이미지 패럴랙스 효과 (미세한 움직임)
- **Smooth Scroll**: 부드러운 스크롤 이동 (CSS scroll-behavior: smooth)
- **Preloader**: 로딩 화면 페이드 인/아웃

## 💻 기술 스택 (간단 요약)

### 프론트엔드
- **프레임워크**: Next.js 16.1.6 (React 19.2.3 기반)
- **언어**: TypeScript 5
- **스타일링**: Tailwind CSS 4 (PostCSS)
- **애니메이션**: 
  - GSAP 3.14.2 (ScrollTrigger, useGSAP)
  - Swiper 12.1.0 (캐러셀)
- **아이콘**: Lucide React 0.563.0
- **폰트**: Geist Sans, Geist Mono (Google Fonts)

### 주요 기능
- **반응형 디자인**: 모바일/태블릿/데스크톱 완벽 대응
- **SEO 최적화**: 메타데이터, Open Graph 설정
- **이미지 최적화**: Next.js Image 컴포넌트 활용
- **부드러운 애니메이션**: GSAP 기반 고성능 애니메이션
- **접근성**: 시맨틱 HTML, ARIA 레이블

### 개발 환경
- **패키지 매니저**: npm
- **린터**: ESLint 9 (Next.js 설정)
- **개발 서버**: Next.js Dev Server
- **빌드 도구**: Next.js Build (Turbopack)
