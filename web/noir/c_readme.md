# NOIR - 프리미엄 웹사이트 프로젝트

> 세련되고 고급스러운 다크 테마 웹사이트를 위한 Next.js 프로젝트 템플릿

---

## 📋 프로젝트 개요

### 브랜드 정보
- **브랜드명**: NOIR (누아르)
- **업종**: 미정 (프리미엄 서비스/제품)
- **컨셉**: 세련됨, 고급스러움, 미니멀리즘
- **타겟**: 프리미엄 서비스를 찾는 고객층

### 프로젝트 상태
이 프로젝트는 **Next.js 16 기반의 초기 설정 상태**입니다. 아직 실제 콘텐츠나 페이지가 구현되지 않았으며, 프리미엄 다크 테마 웹사이트 개발을 위한 기반이 준비되어 있습니다.

### 사이트 목적 (예정)
- 프리미엄 브랜드 이미지 구축
- 세련된 사용자 경험 제공
- 고급스러운 비주얼 커뮤니케이션
- 타겟 고객과의 감성적 연결

## 🔍 섹션별 제안

### 1. Header (헤더)
**목적**: 브랜드 아이덴티티 및 네비게이션

**권장 구성**:
- 로고/브랜드명 (좌측)
- 메인 메뉴 (중앙 또는 우측)
- CTA 버튼 (우측)
- 햄버거 메뉴 (모바일)

**권장 특징**:
- 스크롤 시 배경 변화 (투명 → 반투명 블랙)
- 미니멀한 디자인
- 골드 호버 효과

---

### 2. Hero Section (메인 비주얼)
**목적**: 강렬한 첫인상

**권장 구성**:
- 풀스크린 배경 (영상 또는 이미지)
- 대형 타이포그래피 헤드라인
- 간결한 서브 카피
- CTA 버튼 1-2개

**권장 특징**:
- 100vh 높이
- 텍스트 애니메이션 (페이드인, 슬라이드업)
- 다크 오버레이로 가독성 확보
- 스크롤 다운 인디케이터

---

### 3. About (브랜드 소개)
**목적**: 브랜드 스토리 전달

**권장 구성**:
- 브랜드 철학/비전
- 핵심 가치 (3-4개)
- 브랜드 이미지

**권장 특징**:
- 2열 레이아웃 (텍스트 + 이미지)
- 스크롤 애니메이션
- 타이포그래피 강조

---

### 4. Services/Products (서비스/제품)
**목적**: 제공 가치 소개

**권장 구성**:
- 섹션 제목
- 카드 그리드 (3-4열)
- 각 카드: 아이콘/이미지 + 제목 + 설명

**권장 특징**:
- 호버 시 카드 상승 효과
- 골드 액센트 포인트
- 순차 등장 애니메이션

---

### 5. Gallery/Portfolio (갤러리)
**목적**: 비주얼 쇼케이스

**권장 구성**:
- 그리드 또는 메이슨리 레이아웃
- 고품질 이미지
- 라이트박스 기능

**권장 특징**:
- 호버 시 오버레이 효과
- 이미지 lazy loading
- 필터링 기능 (선택)

---

### 6. CTA (행동 유도)
**목적**: 전환 유도

**권장 구성**:
- 강렬한 헤드라인
- 간결한 설명
- 명확한 CTA 버튼

**권장 특징**:
- 골드 그라데이션 배경
- 대형 버튼
- 마그네틱 효과

---

### 7. Footer (푸터)
**목적**: 연락처 및 부가 정보

**권장 구성**:
- 브랜드 소개
- 연락처 정보
- 소셜 미디어 링크
- 저작권 정보

**권장 특징**:
- 다크 배경
- 미니멀한 레이아웃
- 링크 호버 효과

---

## 🎬 인터랙션 & 애니메이션 (권장)

### 스크롤 애니메이션
- **Fade In**: 섹션이 뷰포트에 들어올 때 페이드인
- **Slide Up**: 요소가 아래에서 위로 슬라이드
- **Parallax**: 배경 이미지 패럴랙스 효과
- **Stagger**: 여러 요소 순차 등장

### 호버 효과
- **버튼**: 골드 그로우 효과, 스케일 확대
- **카드**: 상승 효과, 그림자 강화
- **이미지**: 줌인 또는 오버레이
- **링크**: 골드 컬러 변경

### 특수 효과
- **Magnetic Button**: 마우스 추적 버튼
- **Text Reveal**: 텍스트 3D 회전 등장
- **Smooth Scroll**: 부드러운 스크롤
- **Cursor Follow**: 커스텀 커서 (선택)

## 🎨 브랜드 네이밍: NOIR

**NOIR (누아르)**는 프랑스어로 "검은색"을 의미하며, 다음과 같은 브랜드 가치를 담고 있습니다:

- **세련됨**: 블랙의 우아함과 시크함
- **고급스러움**: 프리미엄 브랜드의 상징
- **미스터리**: 깊이 있는 브랜드 스토리
- **타임리스**: 시대를 초월한 클래식함

이 이름은 다크 테마 웹사이트의 정체성과 완벽하게 일치하며, 프리미엄 서비스/제품을 위한 강력한 브랜드 이미지를 구축할 수 있습니다.
