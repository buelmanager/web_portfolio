## 🔍 섹션별 상세 설명

### 1. Preloader (프리로더)
**목적**: 페이지 로딩 중 브랜드 인상 전달

**구성 요소**:
- 브랜드 로고 애니메이션
- 부드러운 페이드아웃 효과

**특징**:
- 첫 방문 시 브랜드 아이덴티티 각인
- 로딩 완료 후 자연스러운 전환

---

### 2. Header (헤더)
**목적**: 사이트 네비게이션 및 브랜드 아이덴티티

**구성 요소**:
- 로고: "aurora" (au + ro + ra 컬러 강조)
- 메인 메뉴: 홈, 프로그램, 시설, 예약
- CTA 버튼: "예약하기"

**특징**:
- 스크롤 시 배경색 변화 (투명 → 반투명 화이트)
- 모바일 햄버거 메뉴
- 고정 헤더로 항상 접근 가능

---

### 3. Hero Section (메인 비주얼)
**목적**: 첫인상 & 핵심 메시지 전달

**구성 요소**:
- 배경: 스파 테마 영상 (폴백: 고품질 이미지)
- 메인 카피: "자연이 선사하는 깊은 휴식"
- 서브 카피: "aurora Premium Spa & Wellness"
- 강조 정보: "자연 유래 성분 · 프라이빗 케어"
- CTA 버튼: "프로그램 보기", "예약 문의"

**특징**:
- 텍스트 리빌 애니메이션 (단어별 순차 등장)
- 버튼 페이드인 + 블러 효과
- 스크롤 다운 인디케이터 (바운스 애니메이션)
- 다크 그라디언트 오버레이로 가독성 확보

### 8. FAQ (자주 묻는 질문)
**목적**: 고객 문의사항 사전 해결

**구성 요소**:
- 섹션 제목: "자주 묻는 질문"
- 아코디언 형식 (5개 항목)
- 기본 첫 번째 항목 열림

**콘텐츠 목록**:
1. **예약은 어떻게 하나요?**
   - 온라인 예약 시스템 24시간 가능
   - 전화 예약: 10:00~22:00
   
2. **프로그램 소요 시간은 어떻게 되나요?**
   - 60분, 90분, 120분 코스
   - 패키지 이용 시 추가 휴식 시간 제공
   
3. **방문 시 준비물이 필요한가요?**
   - 모든 물품 제공 (가운, 슬리퍼, 어메니티)
   
4. **임산부도 이용 가능한가요?**
   - 임신 16주 이후 산전 마사지 프로그램 가능
   
5. **취소 및 환불 정책은 어떻게 되나요?**
   - 48시간 전: 무료 취소
   - 24시간 전: 50% 수수료
   - 당일/노쇼: 100% 수수료

**특징**:
- GSAP 아코디언 애니메이션
- 호버 시 테두리 색상 변화
- 화살표 회전 애니메이션

## 🎬 인터랙션 & 애니메이션

### 스크롤 애니메이션
- **Fade In**: 모든 섹션 제목 및 카드 요소
- **Text Reveal**: Hero, Features, Product Gallery 등 주요 제목
- **Parallax**: Category Gallery 이미지, CTA 배경 패턴
- **Stagger**: Features 카드 순차 등장 (0.1초 간격)

### 호버 효과
- **버튼**: 배경색 변화, 스케일 확대 (1.05배)
- **카드**: 상승 효과 (translateY -4px), 그림자 강화
- **이미지**: 확대 효과 (scale 1.1), 그라디언트 오버레이

### 특수 효과
- **GSAP ScrollTrigger**: 패럴랙스 스크롤, 이미지 이동
- **Swiper Carousel**: 터치 스와이프, 자동 재생 옵션
- **Blur Animation**: Hero 버튼 등장 시 블러 → 선명
- **Accordion**: FAQ 아코디언 높이 애니메이션

## 💻 기술 스택 (간단 요약)

### 프론트엔드
- **프레임워크**: Next.js 16.1.6 (React 19.2.3 기반)
- **스타일링**: Tailwind CSS 4
- **애니메이션**: GSAP 3.14.2 (ScrollTrigger 포함)
- **캐러셀**: Swiper 12.1.0
- **아이콘**: Lucide React 0.563.0
- **폰트**: Geist Sans, Geist Mono (Google Fonts)

### 주요 기능
- 반응형 디자인 (모바일/태블릿/데스크톱)
- SEO 최적화 (메타데이터, Open Graph)
- 이미지 최적화 (Next.js Image)
- 부드러운 스크롤 애니메이션
- 비디오 폴백 시스템 (영상 로딩 실패 시 이미지 대체)

### 개발 도구
- **빌드 도구**: Turbopack (Next.js 16 기본)
- **패키지 매니저**: Bun
- **타입스크립트**: TypeScript 5
- **린터**: ESLint 9
