# 🌿 aurora | 자연이 선사하는 휴식

> 도심 속 프리미엄 스파 & 웰니스. 자연의 에너지로 몸과 마음의 균형을 찾아드립니다.

---

## 📋 프로젝트 개요

**aurora**는 자연의 치유력을 담은 프리미엄 스파 & 웰니스 센터의 웹사이트입니다. 고요하고 평화로운 자연의 에너지를 시각적으로 표현하며, 방문자에게 깊은 휴식과 힐링의 경험을 전달합니다.

## 📐 페이지 구조

### 1. 🎬 Preloader

- 모바일 반응형 햄버거 메뉴

### 3. 🌅 Hero Section
**자연이 선사하는 깊은 휴식**
- **배경**: 풀HD 비디오 배경 (스파 분위기)

- **메인 카피**:
  - "자연이 선사하는" (애니메이션 텍스트)
  - "깊은 휴식" (애니메이션 텍스트)
  - "aurora Premium Spa & Wellness"
- **서브 카피**: "자연 유래 성분 · 프라이빗 케어"
- **CTA 버튼**:
  - "프로그램 보기" (Primary)
  - "예약 문의" (Secondary)
- **스크롤 인디케이터**: 하단 바운스 애니메이션

### 4. ✨ Features Section
**몸과 마음의 균형을 찾다**

8가지 핵심 프로그램:
1. 🍃 **시그니처 아로마 테라피** - 천연 에센셜 오일로 깊은 이완
2. ✨ **프라이빗 스파 스위트** - 오직 당신만을 위한 독립된 공간
3. 💧 **하이드로 테라피** - 미네랄 워터로 피부 재생
4. 🌙 **명상 & 요가** - 마음의 평화를 찾는 시간
5. ☀️ **페이셜 트리트먼트** - 자연 유래 성분의 프리미엄 케어
6. 🌬️ **바디 스크럽 & 랩** - 전신 디톡스와 보습
7. ❤️ **커플 스파 패키지** - 소중한 사람과 함께하는 힐링

## 🎭 애니메이션 & 인터랙션

### GSAP 애니메이션
1. **TextReveal**: 텍스트 페이드인 + 블러 효과
2. **FadeInSection**: 스크롤 트리거 페이드인 (상/하/좌/우 방향)
3. **ParallaxSection**: 스크롤 시 배경 패럴랙스 효과
4. **StaggerList**: 리스트 아이템 순차 애니메이션
5. **RevealOnScroll**: 스크롤 기반 요소 노출

### 호버 효과
- **hover-lift**: 카드 상승 + 그림자 효과
- **이미지 줌**: 호버 시 scale(1.1) 트랜지션
- **버튼 스케일**: 호버 시 scale(1.05)

### 스크롤 애니메이션
- **ScrollTrigger**: GSAP 스크롤 트리거 플러그인
- **패럴랙스**: 배경 이미지 상하 이동 (yPercent)
- **페이드인**: 뷰포트 진입 시 opacity 0 → 1

### 커스텀 유틸리티
- **text-gradient**: 그라데이션 텍스트 효과
- **glass**: 글래스모피즘 효과 (backdrop-filter)
- **nature-gradient**: 자연스러운 배경 그라데이션
- **organic-border**: 유기적인 곡선 테두리

## 🚀 실행 방법

### 개발 서버 실행
```bash
npm run dev
```
- 로컬 서버: http://localhost:3000
- 핫 리로드 지원

### 프로덕션 빌드
```bash
npm run build
```
- 최적화된 프로덕션 빌드 생성

### 프로덕션 서버 실행
```bash
npm run start
```
- 빌드된 앱 실행

### 코드 린팅
```bash
npm run lint
```
- ESLint 검사 실행
