# NOIR COFFEE - 웹사이트 소개

> 결정적 한 잔의 순간을 담은 프리미엄 커피 브랜드 웹사이트

---

## 📋 프로젝트 개요

### 브랜드 정보
- **브랜드명**: NOIR COFFEE (누아르 커피)
- **업종**: 프리미엄 스페셜티 커피 전문점
- **컨셉**: 미니멀리즘, 본질 추구, 결정적 순간
- **타겟**: 커피의 본질을 이해하고 품질을 중시하는 20-40대 성인

### 사이트 목적
- 브랜드의 철학과 가치를 시각적으로 전달
- 싱글 오리진 원두 제품 소개 및 판매 유도
- 매장 방문 예약 및 고객 경험 제공
- 라이카(Leica) 카메라에서 영감받은 미니멀한 디자인으로 브랜드 아이덴티티 구축

---

## 🎨 디자인 컨셉

### 컬러 팔레트
- **Primary (Black)**: `#000000` - 브랜드의 핵심 색상, 고급스러움과 본질 추구
- **Secondary (White)**: `#FFFFFF` - 깨끗함과 순수함, 미니멀리즘
- **Accent (Noir Red)**: `#C41E3A` - 커피 체리 레드, 포인트 컬러
- **Gray Scale**: `#F5F5F5` ~ `#171717` - 섬세한 그레이 톤으로 깊이감 표현

### 디자인 스타일
- **미니멀리즘**: 불필요한 요소를 제거하고 본질에 집중
- **라이카 감성**: 독일 카메라 브랜드 라이카의 정교함과 장인정신 반영
- **타이포그래피 중심**: Pretendard 폰트로 한글 가독성 최적화, 큰 타이포그래피로 강렬한 인상
- **고급스러운 여백**: 넉넉한 여백으로 시각적 안정감과 프리미엄 느낌 전달

## 🔍 섹션별 상세 설명

### 1. Header (헤더)
**목적**: 사이트 네비게이션 및 브랜드 아이덴티티 강조

**구성 요소**:
- 로고: NOIR 워드마크 + 심볼 마크
- 메인 메뉴: 원두, 메뉴, 공간, 이야기
- CTA 버튼: "방문예약"

**특징**:
- 스크롤 시 배경색 변화 (투명 → 반투명 흰색)
- 모바일 햄버거 메뉴 (전체화면 오버레이)
- 고정 헤더 (sticky header)로 항상 접근 가능
- 미니멀한 디자인으로 콘텐츠에 집중

---

### 2. Hero Section (메인 비주얼)
**목적**: 첫인상 & 브랜드 핵심 메시지 전달

**구성 요소**:
- 배경: 커피 한 잔의 고품질 이미지 (Unsplash)
- 메인 카피: "결정적 / 한 잔" (2줄 구성)
- 서브 카피: "The Decisive Cup"
- 스크롤 인디케이터: "스크롤" 텍스트 + 세로 라인

**특징**:
- 페이지 로드 시 이미지 클립패스 애니메이션 (아래에서 위로 reveal)
- 타이틀 순차 등장 애니메이션 (stagger)
- 스크롤 시 이미지 확대 효과 (parallax scale)
- 스크롤 시 텍스트 페이드아웃 및 위로 이동
- 전체화면 (100vh) 구성으로 강렬한 첫인상

---

### 3. Philosophy (철학)
**목적**: 브랜드 철학과 가치관 전달

**구성 요소**:
- 섹션 제목: 없음 (텍스트가 곧 제목)
- 핵심 메시지:
  - "모든 순간에는 결정적인 한 잔이 있습니다."
  - "우리는 커피의 본질을 추구하며,"
  - "불필요한 것들을 덜어냅니다."
- 사이드 이미지: 커피 원두 클로즈업
- 캡션: "01 철학"

**특징**:
- 단어별 스크롤 애니메이션 (opacity 0.1 → 1)
- 읽는 속도에 맞춘 점진적 reveal 효과
- 이미지 클립패스 애니메이션 (왼쪽에서 오른쪽으로)
- "한 잔이", "본질을" 등 핵심 단어 강조 (accent 클래스)
- 여백을 활용한 고급스러운 레이아웃

---

### 4. Product Showcase (싱글 오리진 원두)
**목적**: 주요 제품 소개 및 구매 유도

**구성 요소**:
- 섹션 라벨: "02 원두"
- 섹션 제목: "싱글 오리진"
- 설명: "산지의 떼루아가 담긴 단일 원산지 원두. 각각의 개성이 온전히 드러납니다."
- 제품 카드 3개 (그리드 레이아웃)
- CTA 링크: "전체 원두 보기" + 화살표 아이콘

**제품 목록**:
1. **에티오피아 예가체프** (ETHIOPIA)
   - 꽃향기와 시트러스의 밝은 산미
   - ₩24,000

2. **콜롬비아 수프리모** (COLOMBIA)
   - 견과류와 카라멜의 균형잡힌 바디
   - ₩22,000

3. **케냐 AA** (KENYA)
   - 와인같은 풍부함과 베리향
   - ₩26,000

**특징**:
- 카드별 순차 등장 애니메이션 (아래에서 위로)
- 스크롤 시 이미지 패럴랙스 효과 (scale 1.1 → 1)
- 호버 시 커스텀 커서 "살펴보기" 표시
- 원산지 라벨 (ORIGIN) 강조
- 가격 정보 명확히 표시

---

### 5. Gallery (갤러리)
**목적**: 브랜드 감성 전달 및 시각적 휴식

**구성 요소**:
- 전체화면 이미지: 커피 내리는 순간 (핸드드립)
- 중앙 인용구: "커피는 순간을 담는 그릇입니다"

**특징**:
- 원형 마스크 애니메이션 (circle clip-path: 15% → 100%)
- 스크롤 시 이미지 확대 효과 (scale 1.3 → 1)
- 미니멀한 타이포그래피로 메시지 강조
- 섹션 번호 없이 자연스러운 흐름

---

### 6. Experience (추출의 순간)
**목적**: 커피 추출 방식 소개 및 경험 제공

**구성 요소**:
- 섹션 라벨: "03 경험"
- 섹션 제목: "추출의 순간"
- 경험 카드 3개 (가로 스크롤)

**경험 목록**:
1. **핸드드립**
   - 바리스타가 직접 내려드리는 정성스러운 한 잔

2. **에스프레소**
   - 압력과 온도의 완벽한 균형

3. **콜드브루**
   - 18시간 저온추출의 깊은 풍미

**특징**:
- 데스크톱: 가로 스크롤 애니메이션 (horizontal scroll)
- 모바일: 세로 스크롤 (일반 레이아웃)
- 각 카드 페이드인 애니메이션
- 호버 시 커스텀 커서 "자세히" 표시
- 핀(pin) 효과로 섹션 고정 후 가로 이동

---

**목적**: 연락처 정보 및 부가 정보 제공

  - **연락**: 02-1234-5678 / hello@noir.coffee
  - **소셜**: 인스타그램, 네이버블로그
- 저작권: © 2026 NOIR COFFEE. 모든 권리 보유.
- 법적 링크: 개인정보처리방침, 이용약관

**특징**:
- 스크롤 시 하단에서 위로 순차 등장 (stagger)
- 3단 그리드 레이아웃 (방문/연락/소셜)
- 링크 호버 시 색상 변화
- 미니멀한 디자인 유지

## 🎬 인터랙션 & 애니메이션

### 스크롤 애니메이션 (GSAP + ScrollTrigger)
- **Fade In**: 모든 섹션 요소가 스크롤 시 페이드인
- **Parallax**: Hero, Gallery 이미지 확대/축소 효과
- **Stagger**: 제품 카드, 경험 카드 순차 등장
- **Clip-path Reveal**: Hero 이미지, Philosophy 이미지, Gallery 원형 마스크
- **Horizontal Scroll**: Experience 섹션 가로 스크롤 (데스크톱)
- **Word Reveal**: Philosophy 섹션 단어별 opacity 변화

### 호버 효과
- **버튼**: 배경색 변화 (투명 → 검정), 텍스트 색상 반전
- **카드**: 커스텀 커서 텍스트 표시 ("살펴보기", "자세히")
- **링크**: 색상 변화 (검정 → 레드)
- **이미지**: 미세한 확대 효과 (scale 1 → 1.05)

### 특수 효과
- **Custom Cursor**: 마우스 커서를 따라다니는 커스텀 커서 (데이터 속성 기반 텍스트 변경)
- **Smooth Scroll**: Lenis 라이브러리로 부드러운 스크롤 경험
- **Header Scroll**: 스크롤 시 헤더 배경 변화 (투명 → 반투명)
- **Mobile Menu**: 전체화면 오버레이 메뉴 (fade + slide)

## 💻 기술 스택 (간단 요약)

### 프론트엔드
- **프레임워크**: React 19 (최신 버전)
- **빌드 도구**: Vite 7 (초고속 빌드)
- **스타일링**: CSS Modules + CSS Variables
- **애니메이션**: GSAP 3.14 + ScrollTrigger
- **스무스 스크롤**: Lenis 1.3

### 주요 기능
- 반응형 디자인 (모바일/태블릿/데스크톱 완벽 대응)
- SEO 최적화 (메타태그, 시맨틱 HTML)
- 이미지 최적화 (Unsplash CDN, WebP 지원)
- 부드러운 스크롤 애니메이션 (60fps 유지)
- 접근성 (ARIA 레이블, 키보드 네비게이션)

### 타이포그래피
- **폰트**: Pretendard Variable (한글 최적화)
- **특징**: 가변 폰트로 다양한 굵기 지원, 웹폰트 최적화

### 배포
- 빌드 도구: Vite
- 패키지 매니저: npm
- 정적 사이트 생성 (SPA)
