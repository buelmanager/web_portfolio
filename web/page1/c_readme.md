# MAISON NOIR - 오뜨 꾸뛰르 패션 하우스 웹사이트

> 파리지앵 럭셔리의 정수를 담은 하이엔드 패션 브랜드 웹사이트

---

## 📋 프로젝트 개요

### 브랜드 정보
- **브랜드명**: MAISON NOIR (메종 누아)
- **업종**: 오뜨 꾸뛰르 패션 하우스 (Haute Couture Fashion House)
- **컨셉**: 영원한 우아함, 에디토리얼 패션, 파리지앵 사부아르 페르
- **타겟**: 럭셔리 패션을 추구하는 하이엔드 고객층
- **설립**: 1987년

### 사이트 목적
- 브랜드의 고급스러운 이미지와 장인정신 전달
- 시그니처 컬렉션 및 서비스 소개
- 프라이빗 상담 예약 유도
- 브랜드 저널을 통한 스토리텔링

## 🎨 디자인 컨셉

### 컬러 팔레트
- **Primary (크림)**: `#F5F3EE` - 부드럽고 따뜻한 아이보리 배경
- **Secondary (차콜)**: `#1A1A1A` - 깊고 세련된 블랙
- **Accent (골드)**: `#907533` - 럭셔리를 상징하는 골드 악센트
- **White**: `#FFFFFF` - 순수함과 우아함
- **Gray**: `#8A8A8A` - 중립적인 회색 톤

### 디자인 스타일
- **타임리스 럭셔리**: 유행을 타지 않는 클래식한 우아함
- **에디토리얼 레이아웃**: 패션 매거진을 연상시키는 비대칭 그리드
- **대담한 여백**: 미니멀하면서도 고급스러운 공간 활용
- **세련된 타이포그래피**: Cormorant Garamond (세리프) + Inter (산세리프) 조합

### 타이포그래피
- **헤드라인**: Cormorant Garamond - 클래식한 세리프체로 우아함 표현
- **본문**: Inter - 현대적인 산세리프체로 가독성 확보

## 🔍 섹션별 상세 설명

### 1. Header (헤더)
**목적**: 브랜드 아이덴티티 및 사이트 네비게이션

**구성 요소**:
- 중앙 로고: "MAISON NOIR"
- 좌측 메뉴: Atelier, Collections, Couture
- 우측 메뉴: Journal, Boutiques, Rendez-vous (상담 예약)
- 모바일 햄버거 메뉴

**특징**:
- 스크롤 시 배경색 변화 (투명 → 반투명 크림)
- 풀스크린 모바일 네비게이션
- 부드러운 페이드 인/아웃 효과

---

### 2. Hero Section (메인 비주얼)
**목적**: 강렬한 첫인상과 브랜드 메시지 전달

**구성 요소**:
- 배경: 풀스크린 패션 이미지 (오뜨 꾸뛰르 드레스)
- 메인 카피: "ETERNAL ELEGANCE REDEFINED"
- 스크롤 인디케이터: "Discover" 텍스트 + 수직 라인
- 그라데이션 오버레이

**특징**:
- 배경 이미지 스케일 애니메이션 (1.2 → 1, 1.5초)
- 타이틀 단어별 등장 애니메이션 (y: 100 → 0, 0.15초 간격)
- 스크롤 시 패럴랙스 효과 (이미지 30% 이동, 타이틀 50% 페이드아웃)

---

### 3. Statement (브랜드 철학)
**목적**: 브랜드의 가치와 장인정신 전달

**구성 요소**:
- 중앙 대형 텍스트: 브랜드 철학 문구
- 떠다니는 이미지 5개: 패션 모델, 런웨이, 아틀리에, 원단 디테일, 꾸뛰르 드레스

**콘텐츠**:
> "Our ATELIER CRAFTS GARMENTS that embody your essence, DEFINING a NEW ERA of COUTURE, ARTISTRY and timeless PARISIAN SAVOIR-FAIRE."

**특징**:
- 단어별 스크롤 기반 페이드인 (opacity: 0.15 → 1)
- 이미지 패럴랙스 (각기 다른 속도와 방향)
- 대문자/소문자 믹스로 시각적 리듬 생성

---

### 4. Featured Collections (시그니처 컬렉션)
**목적**: 주요 컬렉션 소개 및 브랜드 포트폴리오 전시

**구성 요소**:
- 섹션 헤더: "Signature Collections" + "View All Collections" 링크
- 비대칭 그리드 레이아웃 (1 large + 2 normal + 1 wide)

**컬렉션 목록**:
1. **Nuit Éternelle** (영원한 밤) - Autumn/Winter 2026
2. **Lumière de Paris** (파리의 빛) - Spring/Summer 2026
3. **Rêve Blanc** (하얀 꿈) - Bridal Couture
4. **Soirée Dorée** (황금빛 저녁) - Evening Wear

**특징**:
- 카드 등장 애니메이션 (y: 80 → 0, 0.15초 간격)
- 이미지 스크롤 줌 효과 (scale: 1.2 → 1)
- 호버 시 이미지 확대 (scale: 1.05) + 오버레이 다크닝

---

### 5. Services (서비스 소개)
**목적**: 제공 서비스 상세 안내

**구성 요소**:
- 섹션 타이틀: "L'Art du Couture" (꾸뛰르의 예술)
- 소개 문구: 맞춤 오뜨 꾸뛰르부터 프레타포르테까지

**서비스 목록**:

2. **Prêt-à-Porter** (02) - 일상의 우아함을 위한 꾸뛰르 장인정신이 담긴 기성복

4. **Bridal Atelier** (04) - 잊지 못할 순간을 위한 맞춤 웨딩 드레스

**특징**:
- 서비스 아이템 좌측 슬라이드인 (x: -50 → 0, 0.1초 간격)
- 이미지 클립패스 리빌 애니메이션 (inset: 100% → 0%)
- 이미지 패럴랙스 (y: 100 → -100)
