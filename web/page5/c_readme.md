# VELOX - 꿈을 향해 달리다

> 대한민국 프리미엄 전기차 브랜드 웹사이트

---

## 📋 프로젝트 개요

### 브랜드 정보
- **브랜드명**: VELOX (벨록스)
- **업종**: 프리미엄 전기차 제조
- **컨셉**: 포르쉐 스타일의 럭셔리 전기차, 전통과 혁신의 완벽한 균형
- **타겟**: 프리미엄 전기차를 찾는 고객, 퍼포먼스와 디자인을 중시하는 드라이버

### 사이트 목적
- 브랜드 아이덴티티 구축 및 프리미엄 이미지 전달
- 전기차 모델 라인업 소개 (GT-S, GT-E, GT-X)
- 기술력과 퍼포먼스 강조를 통한 브랜드 신뢰도 향상
- 시승 신청 및 구매 상담 유도

## 🔍 섹션별 상세 설명

### 1. Header (헤더)
**목적**: 사이트 네비게이션 및 브랜드 아이덴티티 강화

**구성 요소**:
- 로고: "VELOX" 워드마크
- 메인 메뉴: 모델, 기술, 경험, 브랜드, 구매상담
- 검색 버튼
- 모바일 햄버거 메뉴 (반응형)

**특징**:
- 스크롤 시 배경색 변화 (투명 → 반투명 배경)
- 고정 헤더로 항상 접근 가능
- 모바일에서 풀스크린 네비게이션 메뉴 제공

---

### 2. Hero Section (메인 비주얼)
**목적**: 강렬한 첫인상과 브랜드 핵심 메시지 전달

**구성 요소**:
- 배경: 풀스크린 자동차 주행 영상 (야간 도로)
- 메인 카피: "꿈을 향해 달리다" (대형 한글 타이틀)
- 서브 카피: "DRIVEN BY DREAMS"
- CTA 버튼: "모델 살펴보기", "시승 신청"
- 스펙 바: 0-100 2.8초 | 680마력 | 450km 주행거리
- 스크롤 인디케이터

**특징**:
- 대각선 클립패스 마스크 리빌 애니메이션
- 단어별 순차 등장 효과 (skewY 변형 포함)
- 스크롤 시 비디오 확대 패럴랙스
- 텍스트 페이드아웃 효과

---

### 3. Model Showcase (모델 라인업)
**목적**: 3가지 차량 모델 소개 및 특징 전달

**구성 요소**:
- 섹션 제목: "모델 라인업"
- 비대칭 그리드 레이아웃 (3개 카드)

**모델 목록**:
1. **GT-S** - 순수한 스포츠
   - 0-100km/h 2.8초 | 680마력
   - 다크 골드 악센트

2. **GT-E** - 전기의 미래
   - 0-100km/h 3.2초 | 450km 주행
   - 샴페인 골드 악센트

3. **GT-X** - 모든 길 위에서
   - 0-100km/h 4.1초 | AWD
   - 브론즈 악센트

**특징**:
- 호버 시 악센트 컬러 오버레이 + 이미지 확대
- 대각선 클립패스 리빌 애니메이션
- 이미지 패럴랙스 효과
- 각 카드마다 인덱스 번호 표시 (01, 02, 03)

---

### 4. Performance (퍼포먼스)
**목적**: 압도적인 성능 수치로 기술력 강조

**구성 요소**:
- 배경: 다크 톤 + 자동차 이미지 패럴랙스
- 섹션 제목: "한계를 넘어서다"
- 서브 텍스트: "모든 숫자는 단순한 수치가 아닙니다. 엔지니어링의 결정체입니다."

**성능 스펙 (4개 카드)**:
1. **2.8초** - 0-100km/h | 번개같은 가속
2. **680PS** - 최대 출력 | 압도적인 파워
3. **850Nm** - 최대 토크 | 즉각적인 반응
4. **330km/h** - 최고 속도 | 끝없는 질주

**특징**:
- 숫자 카운터 애니메이션 (0부터 목표값까지 증가)
- 배경 이미지 패럴랙스 (스크롤 시 상하 이동)
- 스펙 카드 순차 페이드인

---

### 5. Horizontal Section (기술 상세)
**목적**: 핵심 기술 4가지를 가로 스크롤로 몰입감 있게 전달

**구성 요소**:
- 섹션 제목: "ENGINEERING - 엔지니어링의 정수"
- 가로 스크롤 슬라이드 (핀 고정)
- 진행률 인디케이터

**기술 슬라이드**:
1. **공기역학** (Aerodynamics) - 바람을 가르는 완벽한 곡선
2. **섀시** (Chassis) - 경량 알루미늄 스페이스프레임
3. **파워트레인** (Powertrain) - 듀얼 모터 전기 구동계
4. **인테리어** (Interior) - 운전자 중심의 콕핏 설계

**특징**:
- 스크롤 시 섹션 고정 후 가로 이동
- 각 슬라이드 내 이미지 패럴랙스
- 인덱스 번호 표시 (01~04)
- 진행률 바 시각화

---

### 6. Heritage (브랜드 역사)
**목적**: 브랜드 스토리와 50년 역사 전달

**구성 요소**:
- 레이아웃: 좌우 분할 (이미지 + 텍스트)
- 섹션 제목: "전통과 혁신의 긴장감"
- 브랜드 철학 텍스트
- 타임라인 (4개 주요 시점)

**타임라인**:
- **1974** - 브랜드 설립
- **1995** - 첫 스포츠카 출시
- **2020** - 전기차 시대 개막
- **2024** - GT-S 글로벌 론칭

**특징**:
- 이미지 대각선 클립패스 리빌
- 텍스트 단어별 페이드인
- "브랜드 스토리" 링크 버튼

---

### 7. Footer (푸터)
**목적**: 뉴스레터 구독 유도 및 사이트맵 제공

**구성 요소**:
- 브랜드 로고 + 태그라인 ("꿈을 향해 달리다")
- 뉴스레터 구독 폼 (이메일 입력)
- 4열 네비게이션:
  - **모델**: GT-S, GT-E, GT-X, 컨셉트카
  - **쇼핑**: 구매 상담, 시승 신청, 견적 내기, 전시장 찾기
  - **서비스**: 서비스센터, 정품 부품, 보증 프로그램, 긴급출동
  - **브랜드**: 회사 소개, 헤리티지, 뉴스룸, 채용
- 소셜 미디어 링크 (인스타그램, 유튜브, 페이스북)
- 저작권 정보 및 법적 링크

**특징**:
- 다크 배경으로 시각적 구분
- 요소별 순차 페이드인 애니메이션

## 🎬 인터랙션 & 애니메이션

### 스크롤 애니메이션
- **Smooth Scroll**: Lenis 라이브러리로 묵직하고 부드러운 스크롤 (duration 1.4초)
- **Fade In**: 모든 섹션 요소가 뷰포트 진입 시 페이드인
- **Parallax**: 배경 이미지 및 비디오 패럴랙스 효과
- **Stagger**: 카드, 텍스트 단어 등 순차 등장
- **Horizontal Scroll**: 기술 섹션 가로 스크롤 (핀 고정)

### 호버 효과
- **버튼**: 그라데이션 배경 fill 애니메이션
- **모델 카드**: 악센트 컬러 오버레이 + 이미지 1.1배 확대
- **링크**: 언더라인 애니메이션

### 특수 효과
- **대각선 클립패스 리빌**: Hero, Model Showcase, Heritage 섹션
- **숫자 카운터**: Performance 섹션 스펙 수치
- **SkewY 변형**: Hero 타이틀 등장 시 기울기 효과
- **진행률 바**: Horizontal Section 스크롤 진행도 시각화

## 💻 기술 스택 (간단 요약)

### 프론트엔드
- **프레임워크**: React 19 (최신 버전)
- **빌드 도구**: Vite 7
- **스타일링**: CSS (CSS Variables 활용)
- **애니메이션**: GSAP 3.14 + ScrollTrigger
- **스무스 스크롤**: Lenis 1.3

### 주요 기능
- 반응형 디자인 (모바일/태블릿/데스크톱 대응)
- 고성능 애니메이션 (GSAP 하드웨어 가속)
- 이미지 최적화 (Unsplash CDN)
- 비디오 배경 (모바일 대응)
- 접근성 고려 (ARIA 레이블, 시맨틱 HTML)

### 개발 환경
- 패키지 매니저: npm
- 린터: ESLint
- 개발 서버: Vite Dev Server (HMR 지원)
