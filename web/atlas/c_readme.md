## 📋 프로젝트 개요

### 브랜드 정보

## 🔍 섹션별 상세 설명

### 1. Preloader (프리로더)
**목적**: 사이트 로딩 중 브랜드 인상 강화

**구성 요소**:
- 로딩 애니메이션
- 브랜드 로고 표시

**특징**:
- 페이지 로딩 완료 시 부드럽게 사라짐
- 첫 방문 시 프리미엄 경험 제공

---

### 2. Header (헤더)
**목적**: 사이트 네비게이션 및 브랜드 아이덴티티

**구성 요소**:
- 로고: "at**las**" (las 부분 골드 컬러 강조)
- 메인 메뉴: 홈, 객실, 다이닝, 스파, 예약
- CTA 버튼: "예약하기"

**특징**:
- 스크롤 시 배경이 반투명 화이트로 변경 (Glass Morphism)
- 모바일에서 햄버거 메뉴로 전환
- 고정 헤더로 항상 접근 가능
- 스크롤 위치에 따라 텍스트 컬러 변경 (투명 배경: 화이트 / 스크롤 후: 다크)

---

### 3. Hero Section (메인 비주얼)
**목적**: 첫인상 & 핵심 메시지 전달

**구성 요소**:
- 배경: 럭셔리 호텔 리조트 영상 (Pexels 동영상) 또는 이미지 폴백
- 메인 카피: "완벽한 휴식의 시작"
- 서브 카피: "atlas Experience"
- 통계 정보: "Luxury Hotel & Resort Since 1920", "100년 전통", "전세계 50개 호텔"
- CTA 버튼: "객실 보기", "예약하기"

**특징**:
- 풀스크린 비디오 배경 (자동 재생, 루프, 음소거)
- 비디오 로딩 실패 시 고해상도 이미지로 자동 전환
- TextReveal 애니메이션으로 텍스트 순차 등장
- TypewriterText 효과로 "atlas Experience" 타이핑 연출
- 하단 스크롤 다운 아이콘 (바운스 애니메이션)
- 다크 그라데이션 오버레이로 텍스트 가독성 확보

### 7. Category Selector (메뉴 구성)
**목적**: 상세 메뉴 정보 제공 및 예약 유도

**구성 요소**:
- 섹션 제목: "메뉴 구성"
- 카테고리 탭: 런치, 디너, 와인, 디저트
- 좌측: 메뉴 이미지 (4:3 비율)
- 우측: 메뉴 상세 정보 (타입, 이름, 설명, 가격, 예약 버튼)
- 하단: 썸네일 네비게이션

**콘텐츠 예시**:
- **런치 A 코스** (5 코스) - ₩89,000
- **시그니처 코스** (8 코스) - ₩250,000
- **와인 페어링** (5 글라스) - ₩90,000~
- **시그니처 디저트** (파티시에 추천) - ₩35,000

**특징**:
- 탭 클릭 시 카테고리 전환
- 썸네일 클릭 시 메뉴 전환
- 좌우 분할 레이아웃 (모바일: 세로 스택)
- 페이드인 애니메이션 (좌측: left, 우측: right)
- 활성 탭 및 썸네일 골드 컬러 강조

## 🎬 인터랙션 & 애니메이션

### 스크롤 애니메이션
- **Fade In**: 모든 섹션이 스크롤 시 페이드인 (FadeInSection 컴포넌트)
- **Text Reveal**: 제목 텍스트가 순차적으로 나타남 (TextReveal 컴포넌트)
- **Typewriter**: Hero 섹션 "atlas Experience" 타이핑 효과 (TypewriterText 컴포넌트)
- **Stagger**: Features 카드가 순차적으로 등장 (delay 증가)

### 호버 효과
- **버튼**: 배경색 변화, 확대 효과 (scale-105)
- **카드**: 상승 효과 (hover-lift), 그림자 증가
- **이미지**: 확대 효과 (scale-110), 오버레이 변화
- **텍스트**: 골드 컬러 전환

### 특수 효과
- **Glass Morphism**: 헤더 스크롤 시 반투명 배경 + 블러
- **Gradient Overlay**: 비디오/이미지 위 다크 그라데이션
- **Smooth Scroll**: 부드러운 스크롤 이동
- **Custom Cursor**: 마우스 커서 커스터마이징

## 💻 기술 스택 (간단 요약)

### 프론트엔드
- **프레임워크**: Next.js 16.1.6 (React 19.2.3 기반)
- **스타일링**: Tailwind CSS 4
- **애니메이션**: GSAP (@gsap/react 2.1.2), 커스텀 애니메이션 컴포넌트
- **캐러셀**: Swiper 12.1.0
- **아이콘**: Lucide React 0.563.0
- **폰트**: Geist Sans, Geist Mono (Google Fonts)

### 주요 기능
- **반응형 디자인**: 모바일/태블릿/데스크톱 완벽 대응
- **SEO 최적화**: 메타데이터, Open Graph 설정
- **이미지 최적화**: Next.js Image 컴포넌트 사용
- **부드러운 애니메이션**: GSAP 및 커스텀 애니메이션 컴포넌트
- **비디오 폴백**: 비디오 로딩 실패 시 이미지 자동 전환
- **접근성**: 시맨틱 HTML, ARIA 레이블

### 개발 환경
- **언어**: TypeScript 5
- **빌드 도구**: Next.js (Turbopack 지원)
- **패키지 매니저**: npm/yarn/pnpm/bun
- **린트**: ESLint 9 (Next.js 설정)
