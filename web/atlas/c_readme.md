# 🏨 atlas - 완벽한 휴식의 시작

> 세계 최고의 럭셔리 호텔 경험. 프리미엄 스위트, 미슐랭 다이닝, 스파 & 웰니스.

---

## 📋 프로젝트 개요

**atlas**는 100년 전통의 럭셔리 호텔 브랜드를 위한 프리미엄 웹사이트입니다. 
전세계 50개 호텔을 운영하며 Forbes 5-Star 인증을 받은 최고급 호텔 체인의 온라인 경험을 제공합니다.

### 🎯 브랜드 정보
- **브랜드명**: atlas
- **슬로건**: 완벽한 휴식의 시작
- **설립**: 1920년 (100년 전통)
- **규모**: 전세계 50개 호텔
- **인증**: Forbes 5-Star
- **키워드**: 럭셔리호텔, 5성급, 리조트, 스위트, 스파, 미슐랭 다이닝

## 🎭 애니메이션 & 인터랙션

### GSAP 애니메이션

#### 1. TextReveal
- **효과**: 글자별 3D 회전 + 페이드인
- **트리거**: ScrollTrigger (화면 85% 진입 시)
- **속성**: opacity, y, rotateX
- **설정**: stagger 0.03초, duration 0.8초

#### 2. FadeInSection
- **방향**: up, down, left, right
- **효과**: 방향별 이동 + 페이드인
- **트리거**: ScrollTrigger (화면 85% 진입 시)
- **설정**: duration 1초, ease 'power3.out'

#### 3. TypewriterText
- **효과**: 타이핑 효과 + 커서 깜빡임
- **설정**: 속도 조절 가능 (기본 100ms)
- **커서**: 선택적 표시 (animate-pulse)

### CSS 애니메이션

#### 1. hover-lift
```css
transform: translateY(-4px)
box-shadow: 0 20px 40px rgba(201, 168, 108, 0.15)
```

#### 2. 이미지 호버
```css
transform: scale(1.1)
transition: 500ms
```

#### 3. 스크롤 인디케이터
```css
animate-bounce (Tailwind)
```

### Swiper 캐러셀
- **네비게이션**: 좌우 화살표 (골드 컬러)
- **페이지네이션**: 점 표시 (골드 활성화)
- **반응형**: breakpoints 설정
- **효과**: 부드러운 슬라이드 전환
